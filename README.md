# carbon-tracker

I want to track my carbon footprint. Every day I want to input from a list of common activities most people do every day labelled with estimated carbon emissions. Then I want to review my emissions daily and weekly, comparing to the global average.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/carbon-tracker.git
cd carbon-tracker
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
